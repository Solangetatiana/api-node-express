# API Node 

Author: Felipe Chan

## Prerequisites

* [NodeJs](https://nodejs.org/) - Javascript
* [Express](http://expressjs.com/) - Framework for NodeJs
* [Mongodb](https://www.mongodb.com/) - Data Base


## Acknowledgments

* NodeJs
* Mongoose
* Express

### Install dependencies

```bash
$ npm install
```

### Run API on local environment

```bash
$ npm start
```

API will be listening at 

```bash
$ http://localhost:3000
```

### Tests

Unity Test
```bash
$ npm run test
```
